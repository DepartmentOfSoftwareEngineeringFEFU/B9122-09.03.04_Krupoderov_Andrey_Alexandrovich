# 📐 Конструктор задач по векторной алгебре

Веб-приложение для преподавателей, позволяющее **создавать задачи по векторам на координатной плоскости**, автоматически **генерировать варианты** с рандомизированными данными и **собирать контрольные / самостоятельные работы** с экспортом в PDF.

> Дипломный проект. Frontend на React + TypeScript, backend на Flask.

---

## ✨ Возможности

- 🧮 **Конструктор задач** — создание задач с векторами и точками на интерактивной координатной плоскости (Konva).
- 🎲 **Генерация вариантов** — автоматическая рандомизация координат в заданных диапазонах.
- ➕ **Операции над векторами** — сложение, вычитание, умножение на скаляр, длина вектора, скалярное произведение (с автоматическим вычислением ответа).
- 📚 **Работы** — сборка контрольных и самостоятельных работ из набора задач.
- 📄 **Экспорт в PDF** — выгрузка задач и вариантов (jsPDF).
- 💾 **Сохранение в БД** — задачи и работы хранятся на backend (Flask + SQLite).

## 🚀 Установка и запуск

Для работы приложения нужно запустить **backend** и **frontend** одновременно (в двух терминалах).

### 1. Backend (Flask)

```bash
cd backend

# создаём виртуальное окружение
python -m venv venv

# активируем его
# Windows:
venv\Scripts\activate
# macOS / Linux:
source venv/bin/activate

# устанавливаем зависимости
pip install -r requirements.txt

# запускаем сервер (http://127.0.0.1:5000)
python app.py
```

База данных SQLite (`instance/database.db`) создаётся автоматически при первом запуске.

### 2. Frontend (React + Vite)

В новом терминале, в корне проекта:

```bash
# устанавливаем зависимости
npm install

# запускаем dev-сервер (http://localhost:5173)
npm run dev
```

Открой в браузере адрес, который покажет Vite (обычно `http://localhost:5173`).

## 🔌 API-эндпоинты

| Метод  | Путь                              | Описание                       |
|--------|-----------------------------------|--------------------------------|
| GET    | `/api/tasks`                      | Получить все задачи            |
| POST   | `/api/tasks`                      | Создать / обновить задачу      |
| DELETE | `/api/tasks/<id>`                 | Удалить задачу                 |
| GET    | `/api/works`                      | Получить все работы            |
| POST   | `/api/works`                      | Создать работу                 |
| DELETE | `/api/works/<id>`                 | Удалить работу                 |
| GET    | `/api/works/<id>/variants`        | Варианты работы                |
| POST   | `/api/works/<id>/variants`        | Сохранить варианты работы      |
| GET    | `/api/health`                     | Проверка состояния сервера     |
